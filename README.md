## Projects

### RosterReady

An automated scheduling platform for businesses to generate employee rosters based on availability, preferences, labour constraints, role coverage, and rest-period rules.

🔗 **Live Site:** [RosterReady](https://rosterready1.vercel.app/)  
📁 **GitHub Repo:** [RosterReady Repo](https://github.com/KacperA02/RosterReady)

Key features:

- Automated employee shift scheduling using a constraint satisfaction algorithm
- Role-based access control for employer and employee workflows
- RESTful APIs for authentication, availability, schedules, and user management
- MySQL relational database modelled with SQLAlchemy
- Schedule locking to preserve selected shift assignments during regeneration
- AWS-based backend and database deployment
- Real-time schedule updates using WebSockets

Technologies:

**FastAPI, Python, React, TypeScript, MySQL, AWS, SQLAlchemy, Tailwind CSS**

### AI-Assisted Assessment Design System

A university project focused on helping lecturers generate, classify, and review assessment questions using GenAI and Bloom’s Taxonomy.

Key features:

- Generate questions from uploaded learning material
- Classify generated questions by Bloom’s Taxonomy level
- Compare target level against predicted level
- Show confidence scores and match status
- Support lecturer review before approval
- Check rubric consistency
- Support auditability and human-in-the-loop review

Technologies:

**FastAPI, React, TypeScript, Python, SQLModel, Transformers, Docker**

A competitive gaming platform for fair team formation, matchmaking, match history, and gaming statistics.

🔗 **Live Site:** [Clash Platform](https://clash-d9110.web.app/)
